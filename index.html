Built for personal use,
not enterprise bloat.

Every feature exists because it's useful. Every line of code earns its place.

🤖

16 LLM Providers

OpenRouter, Anthropic, OpenAI, DeepSeek, Gemini, Groq, MiniMax, Moonshot, Zhipu, DashScope, vLLM, GitHub Copilot, and more. Switch models with one config line.

💬

10 Chat Channels

Telegram, Discord, WhatsApp, Slack, Email, Matrix, Feishu, DingTalk, QQ, and Mochat. Talk to your agent from anywhere.

🔧

MCP Tool Servers

Connect any MCP server — stdio or HTTP transport. Auto-discovers tools on startup. Compatible with Claude Desktop and Cursor configs.

🧠

Memory & Context

SQLite fact store with vector embeddings, RAG retrieval, salience-based decay, and context budget enforcement. Remembers across sessions.

🎯

Skills System

8 built-in skills: clawhub, cron, github, memory, skill-creator, summarize, tmux, weather. Auto-creates new skills from novel tasks.

👥

Subagent Pool

Parallel background execution with configurable concurrency, token budgets, time limits, and inbox review after each subgoal.

⏰

Scheduling

Cron scheduler with expressions and intervals. Heartbeat engine for proactive trigger-driven wake-ups every N minutes.

🛡️

Security Layer

Command sandboxing, prompt injection detection, audit logging, workspace restrictions, and confirmation gates for dangerous operations.

📡

Observability

OpenTelemetry tracing to file or OTLP endpoint, Prometheus metrics port, configurable privacy controls for tool args and memory.
