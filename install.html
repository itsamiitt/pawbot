1. Prerequisites

You need two things before installing:

Python 3.11+

python3 --version

If the output shows 3.10 or lower, upgrade:

macOS brew install python@3.12
Ubuntu / Debian sudo apt install python3.12 python3.12-venv
Windows Download from python.org — tick "Add to PATH"
Any system curl https://pyenv.run | bash && pyenv install 3.12

An LLM API Key

OpenRouter (recommended) — one key gives access to Claude, GPT-4, Gemini, DeepSeek, and more. Free tier available.
→ Get a key at openrouter.ai/keys

You can also use Anthropic, OpenAI, or DeepSeek directly. The setup wizard will ask.

2. Install Methods

⭐ Method 1 — One-liner from website

Fastest. Checks Python, installs Pawbot, runs interactive setup.

curl -fsSL https://pawbot.thecloso.com/install | bash

Method 2 — One-liner from GitHub

Same script, served directly from the repo. Always latest.

curl -fsSL https://raw.githubusercontent.com/HKUDS/pawbot/main/install/setup.sh | bash

Method 3 — pip

Standard PyPI install. Run onboard separately.

pip install pawbot-ai pawbot onboard --setup

Permission denied? Use pip install --user pawbot-ai

Method 4 — uv (isolated environment)

Modern Python package manager. Zero conflicts with other projects.

pip install uv uv tool install pawbot-ai pawbot onboard --setup

Method 5 — Clone & inspect

Read the code first. Also installs in dev mode for contributors.

git clone https://github.com/HKUDS/pawbot.git cat pawbot/install/setup.sh # review bash pawbot/install/setup.sh

Method 6 — Docker

Containerized deployment with Docker Compose.

# First-time setup docker compose run --rm pawbot-cli onboard # Start gateway docker compose up -d pawbot-gateway # Send a message docker compose run --rm pawbot-cli agent -m "Hello!"

Confirm the install

pawbot --version

You should see pawbot 0.1.4.post3. If you get "command not found", see Troubleshooting.

3. First-Time Setup

The interactive wizard creates your workspace, config, and asks for your API key:

pawbot onboard --setup

What it creates:

  • ~/.pawbot/ — your data directory
  • ~/.pawbot/config.json — settings file
  • ~/.pawbot/workspace/ — agent workspace with SOUL.md, USER.md, MEMORY.md

Or configure manually

Run pawbot onboard (without --setup), then edit the config:

# macOS / Linux nano ~/.pawbot/config.json # Windows notepad %USERPROFILE%\.pawbot\config.json

Minimal config — replace the placeholder with your real key:

{ "providers": { "openrouter": { "apiKey": "sk-or-v1-your-key-here" } }, "agents": { "defaults": { "model": "anthropic/claude-sonnet-4-5" } } }
Tip: You can also use an environment variable instead of editing the file:
export PAWBOT_PROVIDERS__OPENROUTER__API_KEY="sk-or-your-key"

4. Verify It Works

pawbot agent -m "Say hello!"

You should see a response from the AI within a few seconds. If you do — Pawbot is fully installed and working.

5. Connect Telegram

Setup takes about 3 minutes.

  1. Message @BotFather on Telegram → send /newbot → copy the bot token
  2. Message @userinfobot → copy your numeric user ID
  3. Add to ~/.pawbot/config.json:
"channels": { "telegram": { "enabled": true, "token": "7123456789:AAHdq...", "allowFrom": ["123456789"] } }
  1. Start the gateway: pawbot gateway
Security: allowFrom restricts who can message your bot. Only listed user IDs are allowed.

Keep it running in the background:

nohup pawbot gateway > ~/.pawbot/logs/gateway.log 2>&1 &

6. Connect WhatsApp

Prerequisite: Node.js 18+ required. Check: node --version

WhatsApp needs two terminals running simultaneously:

Terminal 1 — Link your account

pawbot channels login

A QR code appears. On your phone: WhatsApp → ⋮ → Linked Devices → Link a Device → scan the QR. Keep this terminal open.

Terminal 2 — Start the gateway

pawbot gateway

Add your phone number to config (international format with +):

"channels": { "whatsapp": { "enabled": true, "allowFrom": ["+1234567890"] } }

7. Scheduled Tasks

# Every morning at 9am pawbot cron add --name "morning" --message "Good morning!" --cron "0 9 * * *" # Every hour pawbot cron add --name "hourly" --message "Check status" --every 3600 # Every Monday at 8am pawbot cron add --name "weekly" --message "Weekly summary" --cron "0 8 * * 1" # List / remove pawbot cron list pawbot cron remove <job_id>

Cron expressions

0 9 * * * Every day at 9:00 AM
0 8 * * 1 Every Monday at 8:00 AM
*/30 * * * * Every 30 minutes
0 0 * * * Every day at midnight
0 9,17 * * * 9 AM and 5 PM daily

8. Config Reference

Full ~/.pawbot/config.json example:

{ "agents": { "defaults": { "model": "anthropic/claude-sonnet-4-5", "provider": "auto", "maxTokens": 8192, "temperature": 0.1, "maxToolIterations": 40, "memoryWindow": 100 } }, "providers": { "openrouter": { "apiKey": "sk-or-v1-..." } }, "channels": { "telegram": { "enabled": true, "token": "BOT_TOKEN", "allowFrom": ["USER_ID"] } }, "tools": { "web": { "search": { "apiKey": "BSA-..." } }, "exec": { "timeout": 60 }, "restrictToWorkspace": false } }

Common model choices

anthropic/claude-sonnet-4-5 OpenRouter Best balance of speed & quality
anthropic/claude-opus-4-5 OpenRouter Most capable, slower
anthropic/claude-haiku-4-5 OpenRouter Fastest, cheapest
deepseek-chat DeepSeek Very capable, low cost
openai/gpt-4o OpenRouter GPT-4o via OpenRouter
gemini-2.0-flash Gemini Google Gemini direct

9. Upgrading

# pip pip install --upgrade pawbot-ai # uv uv tool upgrade pawbot-ai # cloned repo cd ~/pawbot && git pull && pip install -e . # or re-run the installer (safe — detects existing install) curl -fsSL https://pawbot.thecloso.com/install | bash

Your ~/.pawbot/config.json and data are never touched during upgrades.

10. Uninstall

# Remove the command pip uninstall pawbot-ai # or: uv tool uninstall pawbot-ai # Remove personal data (config, memory, workspace) rm -rf ~/.pawbot
Backup first: Your conversations and memory live in ~/.pawbot/.

11. Troubleshooting

pawbot: command not found

pip show pawbot-ai | grep Location # Add to PATH (pip --user installs) echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.bashrc source ~/.bashrc

API key error / authentication failed

Check ~/.pawbot/config.json. The key must start with sk-or- (OpenRouter), sk-ant- (Anthropic), or sk- (OpenAI). Must not contain xxx or YOUR_.

WhatsApp QR code expires

It refreshes every 20 seconds. Press Ctrl+C and run pawbot channels login again.

Telegram bot not responding

  1. Check gateway is running: pawbot status
  2. Confirm your user ID is in allowFrom — must be a number, not username
  3. Make sure the bot token matches exactly what BotFather gave you

Config file broken (JSON syntax error)

# Validate python3 -c "import json; json.load(open('$HOME/.pawbot/config.json')); print('OK')" # Regenerate from scratch mv ~/.pawbot/config.json ~/.pawbot/config.json.bak pawbot onboard

Common mistakes: trailing comma after last item, single quotes, comments (//).

ModuleNotFoundError: No module named 'pawbot'

Multiple Python versions on the system. Check pip --version and which pawbot — ensure they use the same Python.
