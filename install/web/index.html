INSTALLATION

Three ways in.

RECOMMENDED
$ curl -fsSL pawbot.thecloso.com/install | bash
GITHUB DIRECT
$ curl -fsSL raw.githubusercontent.com/AiWaldoh/pawbot/main/install/setup.sh | bash
INSPECT FIRST
$ git clone https://github.com/AiWaldoh/pawbot
$ bash pawbot/install/setup.sh
Also available: pip install pawbot-ai
pawbot
REQUIRES
Python 3.11 or higher
Platform macOS · Linux · Windows (WSL)
API key OpenRouter (free) · Anthropic · OpenAI
Node.js 18+ (only for WhatsApp channel)
